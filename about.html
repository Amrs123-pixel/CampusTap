About CampusTap

CampusTap is a thesis capstone project designed to modernize school transportation through a fully digital tap-to-ride system. It integrates RFID identification, real-time GPS tracking, and cloud-based account management to create a safer, faster, and more efficient commuting experience for students. Each tap automatically updates the student’s ride record and balance, eliminating the need for manual fare collection.

The system combines Arduino-powered RFID hardware with a Firebase backend that handles authentication, user data, balance management, and ride tracking. The web application is built using HTML, CSS, and JavaScript, providing students with a clean and responsive interface for checking their balance, viewing their ride history, and monitoring real-time bus locations. Administrators can manage accounts, top up balances, and monitor system activity.

This project demonstrates the practical application of embedded systems, wireless communication, and cloud technologies in creating a reliable and scalable transportation solution. CampusTap aims to improve transparency, convenience, and safety while introducing an innovative cashless transportation method for academic institutions.
