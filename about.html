About CampusTap

CampusTap is a thesis project developed to modernize school transportation and payment systems through the integration of RFID technology, GPS tracking, and cloud-based data management. The system provides students with a convenient tap-to-ride experience, automatically deducting fares from their digital balance while storing ride history in a secure cloud database.

Built using Arduino-based RFID hardware paired with Firebase Authentication and Firestore Database, CampusTap allows seamless user registration, secure login, real-time data updates, and accurate account management. Administrators can monitor student activity, top up balances, and manage accounts through a dedicated web dashboard.

This thesis showcases the practical application of both hardware and software technologies to create an efficient, reliable, and cashless transportation solution for schools. CampusTap aims to improve safety, transparency, and convenience for students and provide administrators with powerful tools for monitoring and system management.
